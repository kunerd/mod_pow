# mod_pow  [![Build Status](https://travis-ci.org/kunerd/mod_pow.svg)](https://travis-ci.org/kunerd/mod_pow)

Extension crate for fast modular exponentiation.

Uses the LR-k-ary algorithm as described in the "[Handbook of Applied Cryptography](http://cacr.uwaterloo.ca/hac/about/chap14.pdf), 14.82".
